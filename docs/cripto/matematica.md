# Fundamentos de Criptografia

O algoritmo RSA é um dos pilares da criptografia assimétrica. A encriptação de uma mensagem $m$ para obter o texto cifrado $c$ é feita através da seguinte fórmula modular:

$$c \equiv m^e \pmod{n}$$

Onde $e$ é o expoente público e $n$ é o produto de dois números primos gigantescos.
